Detailed Description

Contains functions performing each step of the algorithm. The calculation of Eigen Values, Eigen Vectors and integrated support degree score has been implemented using an external library called gsl.

Calculates the integrated support degree score of all sensors at a specific time stamp.

Produces an array of integrated support degree scores where the score nth sensor is the nth element in that array.

Parameters
[in]sensorinputsReadings of all sensors at a specific timestamp.
[in]list_of_alphasContribution rates a.k.a alphas where the contribution rate of nth sensor is the nth element in that array.
[in]list_of_phiAccumulated contribution rates a.k.a phis where the accumulated contribution rate of m sensors is the mth element in array.
[in]dmatrixSupport Degree Matrix of the sensor readings at a specific time stamp.
[in]criterionThe minimum value of accumulated contribution rate required to determine the number of principal components to be taken into consideration. The expected value is not in percentage but between 0 and 1.
[in]sizeThe number of sensors.
Returns
An array consisting of the integrated support degree score of each sensor at a specific timestamp.

Produces EigenVectors for a specific EigenValue of Support Degree Matrix.

Creates an 1D array of EigenVectors for a given support degree matrix using functions from the gsl external library. The actual EigenValue is not supplied but the position of EigenValue in a descending sorted array of EigenValues as returned by eigen_value_calculation method is given.

Parameters
[in]dmatrixThe Support Degree matrix produced by sdm_calculator function.
[in]sizeThe number of sensors being considered which is same as size of dmatrix.
[in]columnThe column number of the EigenVector matrix produced by the gsl library. This is same as the position of EigenValue when the EigenValues are sorted in descending order.
Returns
A 1D array of EigenVectors for the given EigenValue.
